# Synthesis

Cross-cutting theme pages that span multiple books. A synthesis page asks a big
question and documents what the whole wiki thinks — agreements, disagreements,
and an evolving thesis.

**Naming convention:** `<theme-slug>.md`
Examples: `what-makes-a-great-leader.md`, `trust-vs-accountability.md`,
`managing-up-vs-managing-down.md`
