# Situations

**This is the core query layer.** Each page answers a specific leadership situation
using everything the wiki has accumulated from ingested books.

Ask: *"How should I handle X?"* — the LLM searches here first, then backs answers
with concepts and books. Good query answers are written back here so they compound.

**Naming convention:** `<situation-slug>.md`
Examples: `team-in-conflict.md`, `low-performer.md`, `new-team-first-90-days.md`,
`giving-difficult-feedback.md`, `burned-out-team.md`, `stakeholder-misalignment.md`
